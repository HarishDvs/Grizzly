# Example Codex — Treasure Island, Part One

A working demonstration of the Grizzly codex format, built from Robert Louis
Stevenson's *Treasure Island* (1883, public domain), Part One: "The Old Buccaneer"
(chapters 1-6).

Demo subset: the index covers all six chapters; full cards are provided for
chapters 1-2, one arc, and one character, which is enough to show every template
in use. A real codex cards everything.

Note how the cards stay **extractive**: events, objects, exact words, chapter
citations. No interpretation ("Bones is a tragic figure"), no speculation. The
manuscript stays the source of truth; the cards are its index.
